# 内容审核清单

供维护者审查新增或大幅修改的文档、案例与代码示例。

## 文档（mds/）

- [ ] 文件名符合 [文档命名规范](文档命名规范.md)
- [ ] 含 `**关键词:**` 与 `**标签:**`（叶子文档）
- [ ] 简介说明「是什么 / 在哪用」
- [ ] 技术表述准确，无过时版本号（或已标注适用范围）
- [ ] 内部链接有效，图片路径存在且无 `?raw=true`
- [ ] AI Coding 指南（若适用）含交互 / 方法 / 应用 / 提示词范例
- [ ] `python tools/check/check_docs.py` 通过

## 实战案例（cases/）

- [ ] 含背景、症状、排查、根因、方案、效果、经验教训
- [ ] 有「图谱知识点映射」回链
- [ ] 数据与日志示例已脱敏

## AI 对话（ai-cases/）

- [ ] Prompt 用 `>` 引用块包裹
- [ ] 含追问 / 修正过程，非仅最终答案
- [ ] 有「关键收获」方法论总结

## 代码示例（code/）

- [ ] 可独立运行，README 说明构建方式
- [ ] 路径符合 `code/{模块}/{文档编号}/{示例名}/`
- [ ] 无密钥、无环境相关绝对路径

## 质量评分（参考）

| 等级 | 标准 |
|------|------|
| A | 结构完整 + 自检通过 + 有配套延伸资源 |
| B | 结构完整 + 自检通过 |
| C | 内容有用但缺标签/链接等待补 |
| D | 需重写或拒绝合并 |

## 定期审查

- 每季度：抽查 10% 叶子文档外链有效性
- 大版本游戏引擎发布后：更新相关引擎/工具文档
- 收到 Issue 反馈后：72h 内 triage，合理者纳入迭代
