## 三、积累实践经验

做一些 Demo 或参加实习，让别人看到你做出来的可视成果。

| 实践 | 说明 |
|------|------|
| [Unity Open Projects](https://unity.com/open-projects) | 开源协作项目，适合 Unity 新人 |
| [Unity Learn](https://learn.unity.com/) | 官方免费课程 |
| 自研小 Demo | 移动 + UI + 本地存档，参考 [3.3.2 角色](../3.研发能力/3.3.2.角色.md) |
