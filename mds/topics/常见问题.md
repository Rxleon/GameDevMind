## 如何在 GitHub 上搜索？

在仓库页面按 **`/`** 聚焦搜索框，或使用顶部 **Search**。

**常用语法：**

| 意图 | 示例 |
|------|------|
| 全仓库搜词 | `网络同步` |
| 限定目录 | `path:mds/3.研发能力 帧同步` |
| 限定文件类型 | `extension:md 对象池` |
| 搜代码示例 | `path:code DrawCall` |
| 搜案例 | `path:cases 内存` |

**技巧：** 先查 [KEYWORDS.md](../../KEYWORDS.md) 找到文档名，再用 `path:mds/文件名关键词` 精确定位段落。

## 如何贡献？

1. 阅读 [CONTRIBUTING.md](../../CONTRIBUTING.md) 与 [文档命名规范](../../docs/文档命名规范.md)
2. 本地运行 `python tools/check/check_docs.py`
3. 提交 Pull Request

维护者审核参考 [内容审核清单](../../docs/内容审核清单.md)。
